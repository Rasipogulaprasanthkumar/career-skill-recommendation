# Career Path & Skill Mapping Recommendation System  

## 📌 Project Overview  
This project is a **Career Path & Skill Mapping Recommendation System** designed to recommend suitable career roles for users based on their skills, courses, and certifications. The system leverages **SQL databases** and **Entity Relationship Diagrams (ERDs)** to manage data efficiently and provide recommendations.  

## 🎯 Features  
- Store and manage **users, skills, courses, certifications, and test results**  
- Recommend suitable **career roles** based on skills and interests  
- Map skills with required courses and certifications  
- Provide feedback for continuous improvement  
- SQL queries for recommendations and reporting  

## 🗂️ Project Structure  
- `schema.sql` → Database schema (tables and relationships)  
- `queries.sql` → SQL queries for recommendations and reports  
- `erd_reference.sql` → ERD reference for relationships  
- `README.md` → Project documentation  

## 🛠️ Tech Stack  
- **Database:** MySQL  
- **Languages:** SQL  
- **Modeling:** ERD (Entity Relationship Diagram)  

## 📊 ERD Overview  
The ERD includes entities such as:  
- **User** (skills, courses, certifications, test results)  
- **Skill** (mapped to courses and roles)  
- **Course & Certifications**  
- **Job Roles** (mapped with required skills)
